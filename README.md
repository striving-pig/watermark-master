# Watermark Master Pro - 水印大师专业版

一款功能强大、界面精美的在线图片水印工具，专为批量处理和专业水印设计而生。支持文字水印和图片水印，提供实时预览、预设管理和批量导出功能。

## 📸 项目简介

Watermark Master Pro 是一个纯前端的图片水印解决方案，无需安装任何软件，直接在浏览器中运行。采用现代化的 UI 设计和直观的操作流程，让水印添加变得简单而高效。

### 核心优势

- 🚀 **零配置启动**：打开即用，无需安装依赖
- 🎨 **专业界面**：采用现代设计语言，操作流畅
- 💾 **本地处理**：所有图片处理在本地完成，保护隐私
- ⚡ **高性能**：基于 Canvas API，处理速度快
- 📱 **响应式设计**：完美适配桌面、平板和移动设备

## ✨ 功能特性

### 1. 水印类型

#### 文字水印
- ✅ 自定义水印文字内容
- ✅ 5 种精选字体（Inter、JetBrains Mono、Arial、Times New Roman、微软雅黑）
- ✅ 字体大小调节（10px - 200px）
- ✅ 自定义填充颜色（支持颜色选择器）

#### 图片水印
- ✅ 支持上传 PNG、JPG、GIF 等格式的水印图片
- ✅ 实时预览水印图片效果
- ✅ 支持透明背景水印图片

### 2. 水印调整参数

| 参数 | 范围 | 说明 |
|------|------|------|
| 透明度 | 0% - 100% | 精确控制水印可见度 |
| 缩放比例 | 0.1x - 5.0x | 自由调整水印大小 |
| 旋转角度 | 0° - 360° | 任意角度旋转 |
| 水平位置 | 0% - 100% | X 轴位置控制 |
| 垂直位置 | 0% - 100% | Y 轴位置控制 |

### 3. 交互功能

- 🖱️ **拖动定位**：在画布上直接拖动水印到目标位置
- 👁️ **实时预览**：所有调整即时反映在画布上
- 📊 **数值显示**：滑块旁实时显示当前数值
- 🔄 **一键重置**：快速恢复默认设置

### 4. 批量处理

- 📦 **批量上传**：支持同时选择多张图片
- 🖼️ **缩略图管理**：已上传图片以网格形式展示
- 🎯 **快速切换**：点击缩略图即可切换当前编辑图片
- 📥 **批量导出**：一键导出所有带水印的图片

### 5. 预设管理系统

- 💾 **保存预设**：将当前水印配置保存为预设
- 🔄 **应用预设**：快速加载已保存的水印配置
- 📂 **本地存储**：预设数据保存在浏览器本地
- 🔢 **多预设支持**：可保存多个不同的水印配置

### 6. 数据持久化

- 自动保存当前水印配置
- 自动记录当前编辑的图片
- 刷新页面后自动恢复工作状态
- 所有数据存储在 localStorage

## 🚀 快速开始

### 在线使用

1. 下载本项目到本地
2. 用浏览器打开 `index.html` 文件
3. 开始使用水印大师！

### 基本操作流程

```
1. 上传图片
   └─ 点击「批量上传图片」按钮
   └─ 选择一张或多张图片

2. 配置水印
   └─ 选择水印类型（文字/图片）
   └─ 调整水印参数
   └─ 在画布上拖动水印定位
   └─ 支持水印全屏平铺

3. 预览效果
   └─ 实时查看水印效果
   └─ 切换其他图片查看效果

4. 导出图片
   └─ 点击「开始批量导出」
   └─ 图片自动下载到本地
```

## � 使用指南

### 文字水印使用步骤

1. 确保选中「文字水印」选项
2. 在「水印内容」输入框中输入水印文字
3. 选择字体（推荐使用 Inter 或微软雅黑）
4. 调整字体大小（建议 30-60px）
5. 选择填充颜色（建议浅色或半透明色）
6. 调整透明度（建议 30%-60%）
7. 在画布上拖动水印到合适位置
8. 完成！

### 图片水印使用步骤

1. 选择「图片水印」选项
2. 点击上传按钮选择水印图片
3. 预览水印图片效果
4. 调整缩放比例（建议 0.5x - 2.0x）
5. 调整透明度（建议 30%-70%）
6. 如需旋转可调整旋转角度
7. 拖动水印到目标位置
8. 完成！

### 预设管理使用

#### 保存预设
1. 配置好水印参数
2. 点击「存为新预设」按钮
3. 预设会自动保存到本地

#### 应用预设
1. 在预设下拉菜单中选择已保存的预设
2. 点击「应用预设」按钮
3. 当前水印配置会被替换为预设配置

## 🔧 自定义开发

### 修改默认配置

在 `index.html` 中找到以下代码段进行修改：

```javascript
let watermark = {
    type: 'text',                    // 默认水印类型
    text: 'Watermark Master',        // 默认水印文字
    font: "'Inter', sans-serif",     // 默认字体
    fontSize: 40,                    // 默认字体大小
    fontColor: '#ffffff',            // 默认字体颜色
    opacity: 0.5,                    // 默认透明度
    scale: 1,                        // 默认缩放比例
    rotation: 0,                     // 默认旋转角度
    x: 0.5,                          // 默认 X 位置
    y: 0.5                           // 默认 Y 位置
};
```

### 添加新字体

1. 在 `<head>` 中添加字体链接
2. 在 `fontSelect` 下拉菜单中添加选项
3. 在 JavaScript 中更新字体配置
